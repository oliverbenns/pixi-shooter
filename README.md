# Pixi Shooter

Experimenting & prototyping with [PixiJS](https://github.com/pixijs/pixi.js).

## Install
- `nvm install`
- `npm install`

## Development
- `npm run develop`
- Open `http://localhost:8080`

## Build
- `npm run build`
- Javascript bundle will be in `/public/assets/bundle.js`

## Github page deployment
- `npm run deploy`
